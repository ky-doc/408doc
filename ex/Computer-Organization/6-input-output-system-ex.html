                                    # 中断优先级

                                    例题 中断响应由高到低的优先次序宜用()。

                                    A.A.A.访管→程序性→机器故障

                                    B.B.B.访管→程序性→重新启动

                                    D.D.D.程序性→I/OI/OI/O→访管

                                    解：BBB。硬件故障中断属于最高级，其次是软件中断，所以机器故障最高。重新启动应该等待其他任务全部完成，所以优先级最低。访管指令是软件指令中最紧急的，所以软件中断中优先级最高。程序性中断要高于重新启动。所以次序为机器故障→访管→程序性→重新启动。
