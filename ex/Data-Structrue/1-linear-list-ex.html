                          # 线性表习题

                          # 基本概念

                          例题 以下()是一个线性表。

                          A.A.A.由nnn个实数组成的集合

                          B.B.B.由100100100个字符组成的序列

                          C.C.C.所有整数组成的序列

                          D.D.D.邻接表

                          解：BBB。线性表定义的要求为：相同数据类型、有限序列。选项CCC的元素个数是无穷个，错误；选项AAA集合中的元素没有前后驱关系，错误；选项DDD属于存储结构，线性表是一种逻辑结构，不要将二者混为一谈。只有选项BBB符合线性表定义的要求。

                          # 顺序表

                          例题 设线性表有nnn个元素，严格说来，以下操作中，()在顺序表上实现要比在链表上实现的效率高。

                          Ⅰ.输出第iii（1⩽i⩽n1\leqslant i\leqslant n1⩽i⩽n）个元素值。

                          Ⅱ.交换第333个元素与第444个元素的值

                          Ⅲ.顺序输出这nnn个元素的值

                          A.A.A.Ⅰ

                          B.B.B.Ⅰ、Ⅲ

                          C.C.C.Ⅰ、Ⅱ

                          D.D.D.Ⅱ、Ⅲ

                          解：CCC。对于Ⅱ，顺序表仅需333次交换操作；链表则需要分别找到两个结点前驱，第444个结点断链后再插入到第222个结点后，效率较低。对于Ⅲ，需依次顺序访问每个元素，时间复杂度相同。

                          # 链表

                          # 结构选择

                          例题 一个链表最常用的操作是在末尾插入结点和删除结点，则选用()最节省时间。

                          A.A.A.带头结点的双循环链表

                          B.B.B.单循环链表

                          C.C.C.带尾指针的单循环链表

                          D.D.D.单链表

                          解：AAA。首先分析要求。末尾删除和插入结点，则需要对尾部操作。BBB是循环链表，但是方向是头结点到尾部，如果操作尾部需要O(n)O(n)O(n)的时间。CCC带了尾指针，所以可以对尾指针直接插入元素，但是不能删除尾指针所指向的最后一个元素，因为是单链表，所以不能找到尾指针的前驱。DDD单链表只能从头开始找，所以不合适。对于AAA。虽然它是带头结点，但是是循环链表，所以头节点的前驱就是尾结点，且是双链表，所以可以向前。若是删除尾结点直接删除头节点的前驱，同理插入尾节点，直接对头节点的前驱操作。

                          # 链表逻辑

                          例题 某线性表用带头结点的循环单链表存储，头指针为head，当 head->next->next=head成立时，线性表长度可能是()。

                          A.0A.0A.0

                          B.1B.1B.1

                          C.2C.2C.2

                          D.D.D.可能为000或111

                          解：DDD。这个题目很容易就会选择BBB。对于BBB，若是有一个元素，则head−>nexthead->nexthead−>next就指向该元素，该元素后面没有了，就指向头节点从而head->next->next=head。若元素有000个，则head->next=head，则head->next->next=head->next=head，所以也成立。
